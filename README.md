# Roguelike-Map-Generator-Tiralabra-2019-II-

### Käyttöohje:

Suoritettavan .jar:in voi generoida komennolla mvn package. Graafisessa käyttöliittymässä voikullekin algoritmille antaa haluamansa parametrit, nappia painamalla kartta luodaan. Enterillä näkee kartan generoitumisen pala palalta, ecillä pääsee takaisin menuun. Täysin absurdeilla parametreilla on mahdollista saada tämä tekemään niin isoja karttoja, että niissä kestää ikuisuus tai ainakin niin kauan että muisti loppuu. Käytä omaa harkintaa.

Kartat on mahdollista ainakin hyvin pienellä muokkauksella yhdistää esim [otm-kurssilla tekemääni roguelikeen](https://github.com/konstakallama/otm-harjoitustyo).
